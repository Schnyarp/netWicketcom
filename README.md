This repository contains the code for the [netWicket.com](https://netwicket.com) domain-for-sale landing page.

## Purpose

This landing page announces that netWicket.com is available for purchase and provides an easy way for interested buyers to contact the domain owner.

## Features

- Modern, mobile-friendly single-page design
- Clearly advertises that netWicket.com is for sale
- **Visitors can make contact with the site using the integrated contact form**
- Inquiry form works with Formspree or your preferred provider for collecting offers and questions
- Automatically detects and displays the current domain name on the page and in form submissions
- Fast, secure, and easy deployment (static HTML, CSS, and JavaScript)
- Simple to customize content and appearance

## Usage

1. **Clone or download this repository.**
2. **Edit the content** in `index.html` as needed (domain name, form endpoint, features, etc.).
3. **Deploy to your preferred static hosting provider** (Netlify, Vercel, GitHub Pages, cPanel, etc.).
4. **Update DNS settings** to point netWicket.com to your hosting provider.

## Customization

- Update the domain name in the header, title, and throughout the page as needed.
- Set the Formspree endpoint or other contact integration for form submissions.
- Adjust design, features, or text to match your preferences or branding.

## DNS Setup

For best results, point netWicket.com to your hosting provider using a CNAME, ALIAS/ANAME, or A record as appropriate.  
Avoid HTTP redirects, as these can interfere with domain detection and inquiry tracking.
